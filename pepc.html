👩🏻‍💻 Demo instructions

You will see empty rectangles unless this browser implementes basic support for the <permission> element. In Chrome M120 or later, follow these instructions to enable experimental support:
  • Shut down the browser process.
  • Restart the browser with these command line flags: 
    --enable-blink-features=PermissionElement --enable-features=PermissionElement
  • Ensure the flags actually took effect by verifying they appear on: 
    chrome://version
